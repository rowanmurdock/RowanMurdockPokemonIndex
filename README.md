# Pokémon Index

A responsive web application designed to browse and display data from the Pokémon universe. 

## Project Overview

The Pokémon Index provides a clean, paginated interface for exploring Pokémon. Application was built to work seamlessly on desktop and mobile.

### API Usage
The PokeAPI is used to get images, names, and IDs of Pokemon. 

## Installation and Local Execution

To run this project on your local machine, follow these instructions:

### 1. Clone the Repository
Download or clone the project files to your local environment.

### 2. Verify File Structure
Ensure the following files are in the same root directory:
* `index.html`
* `pokemon_style.css`
* `app.js`
* `/assets`

### 3. Launch the Application
You can run this project in two ways:
* **Direct Access:** Simply open `index.html` in any modern web browser.
* **GitHub Pages:** Click on the link to view the GitHub Pages deployed Pokemon Index:  https://rowanmurdock.github.io/RowanMurdockPokemonIndex/
